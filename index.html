EXPERIENCIA

JC Telecomunicaciones

2022 - 2023

Desarrollé JC Telecomunicaciones como un proyecto de universidad, un sistema integral de inventario y registro de información de empleados, implementado en Java utilizando NetBeans y MySQL. El software incluye módulos clave como autenticación de usuario, registro y gestión de productos y empleados, así como la generación de informes y listas de inventarios. Mi rol abarcó el diseño de las interfaces, la creación del backend para la autenticación de usuarios y la conexión con la base de datos, además de implementar el registro y consulta de datos para generar informes y manejar el stock de productos.

Mi trabajo incluyó conectar las interfaces con la base de datos y desarrollar funcionalidades para gestionar usuarios, productos y empleados. Además, creé procesos para extraer y mostrar datos en listas e informes automatizados, permitiendo un control eficiente del inventario.

[ Desarrollador FullStack Trainer ]

SICOT

2023 - 2024

SICOT (Sistema Interno de Control Operacional de Tecnología) fue desarrollado para la empresa Porcinos del Alba como parte de mis prácticas universitarias. Este sistema gestiona el control de asistencias técnicas del departamento de tecnología, inventario de equipos informáticos y la generación de informes sobre fallas y requisiciones. También incluye la generación de formatos para requisiciones y memorándums, además de la autenticación y jerarquía de usuarios (administrador y técnico). Me encargué del diseño y desarrollo de la base de datos en MySQL, conexión con el backend y creación de la lógica para el registro de usuarios, equipos y actividades del departamento.

Entre mis responsabilidades estuvo el diseño y construcción de la lógica para la gestión de inventario, formularios de memorándum, y la conexión entre interfaces. También implementé la lógica para el registro de equipos, departamentos, usuarios, empleados por departamentos y estados de los equipos, incluyendo su ubicación dentro de la empresa. El sistema maneja el flujo de datos de manera eficiente, permitiendo un control detallado y preciso de todas las operaciones relacionadas.

[ Desarrollador Backend "Pasantía" ]

LinkCollect

2023 - 2024

Desarrollé esta aplicación web sencilla para guardar y gestionar enlaces (URLs) de páginas y aplicaciones web. Los usuarios pueden registrar sus propios enlaces, visualizarlos en tarjetas que muestran su URL, nombre y descripción, además de poder crear, actualizar y eliminar sus tarjetas. La aplicación cuenta con autenticación de usuarios y cada usuario accede a sus propias tarjetas de forma individual.

Mis responsabilidades incluyeron la conexión con la base de datos MongoDB y el desarrollo del backend usando Node.js y Express. Implementé la lógica para el registro y autenticación de usuarios con la librería Passport, además de desarrollar el CRUD completo y gestionar las rutas de la aplicación. También participé en el diseño de las interfaces con la ayuda de IA.

[ Desarrollador Backend "Jr" ]

Social Sphere

Actualmente

Actualmente, estoy desarrollando esta aplicación web, una red social dirigida a programadores. SocialSphere permitirá a los usuarios registrarse, autenticarse y acceder a un dashboard donde podrán ver y publicar contenido. Los usuarios podrán interactuar con publicaciones mediante likes, comentarios y compartir imágenes con descripciones. Además, contará con chat entre amigos agregados y perfiles de usuario.

Mis responsabilidades abarcan todo el desarrollo backend, incluyendo la lógica de autenticación, registro, inicio de sesión, recuperación de contraseñas y manejo de publicaciones. También he diseñado la estructura de la base de datos y las interfaces, con algo de apoyo de IA. Utilizo Node.js, Express, MongoDB, junto con librerías como Passport, bcrypt, Express-session, y herramientas de pruebas como Jest y Supertest.

Otra responsabilidad es encargarme de realizar el testing completo de la aplicación, implementando pruebas unitarias, de integración y funcionales para garantizar la calidad del sistema. Uso Jest para las pruebas unitarias y de integración, y Supertest para las pruebas funcionales de las rutas y APIs.
